OpenExpertSDR
=============
### Изменения относительно оригинальной версии:
- Изменение интерфейса
- Поддержка плагинов (свой формат)
- Исправление ошибки при воспроизведении IQ файлов
- Изменение формата файлов QuickWave для корректного их воспроизведения в сторонних аудиоплеерах (IEEE FLOAT)
- Добавлена экспериментальная поддержка ExtIO плагинов (только для устройств возвращающих IQ семплы через аудиокарту (hw type == 4))
- Экспериментальные изменения работы с DttSP для уменьшения нагрузки на CPU в режимах ANF/NR/NB (требуется сбросить настройки)
- Оптимизация графики с целью уменьшения нагрузки на CPU
- Поддержка 2х звуковых карт
